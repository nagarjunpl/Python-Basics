# Data Types in Python

Data types specify the type of data a variable can hold in Python.

## 1. Numeric Types

-   **int** -- integers like 5, -10, 200
-   **float** -- decimal values like 3.14, -2.9
-   **complex** -- complex numbers like 2+3j

## 2. String Type

-   **str** -- sequence of characters like "Hello"

## 3. Boolean Type

-   **bool** -- True or False

## 4. Sequence Types

-   **list** -- ordered, mutable, e.g., \[1, 2, 3\]
-   **tuple** -- ordered, immutable, e.g., (1, 2, 3)
-   **range** -- range of numbers, e.g., range(5)

## 5. Mapping Type

-   **dict** -- key-value pairs, e.g., {"name": "Nagarjun"}

## 6. Set Types

-   **set** -- unordered unique values, e.g., {1, 2, 3}
-   **frozenset** -- immutable set

## 7. Binary Types

-   **bytes** -- immutable binary data
-   **bytearray** -- mutable binary data
-   **memoryview** -- memory access to binary data
